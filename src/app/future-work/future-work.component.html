Future Work

Sky is the Limit

Design

  • As it is customed made, any theme can be integrated, depends on end users
  • Can add more colours to the table header, columns, rows using Analogous or Split Complementary Schemes
  • URL or long text can be made more responsive with hide/show, sub-rows, tooltips etc.
  • Show Entry and Search options can be made more attractive by size and highlight on hovering effect
  • Input field, buttons can be converted into dropdown, linked list etc.

Features

  • Dynamic table can be created; app will fetch JSON keys as headers, build number of columns based on the number of JSON properties
  • Other data formats can be used e.g. XML
  • Selecting on a particular row will take user to its own page
  • User can choose/upload other JSON files/URLs
  • Graphs and Charts can be generated integrating D3js, depends on the type on data
  • Custom and individual column filtering input box
  • Advanced search can be created
  • Input validation can be done easily
  • Multiple tables
  • Options to add/remove or show/hide columns
  • Option to add/remove items/rows
