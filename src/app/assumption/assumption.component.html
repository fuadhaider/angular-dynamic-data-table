Assumptions

Technical Stack

  • Angular 6
  • DataTables

Development

  • Used the latest version of npm, Angular and DataTables
  • Integrated DataTables with the project
  • Created table framework on the template with DataTable directive
  • DataTable options and trigger are attached to tempalte table with property binding
  • A service loads JSON file from url and sends it to component
  • Component subscribes JSON data show it in a table
  • Component also decorates the table and its options
